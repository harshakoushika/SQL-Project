# University Database Management System (MySQL)

A relational database project designed to simulate a real-world university management system.  
This project demonstrates database design, relational modeling, and analytical SQL querying using MySQL.

---

## Live Scope

This project focuses on:
- Relational schema design
- Data integrity using constraints
- Business-oriented SQL queries
- Analytical reporting using aggregate functions and joins

---

## Core Entities

- Students
- Faculty
- Departments
- Courses
- Enrollments

The schema models one-to-many and many-to-many relationships using primary and foreign keys to maintain referential integrity.

---

## Features

- Structured database design
- Primary & Foreign Key constraints
- Aggregate reporting queries
- Multi-table joins
- Subqueries for analytical insights
- Real-world business case simulations

---

### Data Definition Language (DDL)

### Querying & Filtering
- WHERE

### Aggregation & Analysis
- COUNT, SUM, AVG, MIN, MAX

- HAVING

### Relational Operations
- INNER JOIN
- LEFT JOIN

## Sample Analytical Queries

- Find total students per department
- Identify second highest salary
- Increase faculty salary by 10%
- Find students enrolled in more than two courses
- Retrieve department-wise average salary
- Identify faculty earning above department average
